Simple IP Info - Support

About the App

Simple IP Info is an iOS app that allows you to look up information about any IP address or domain name, including location, organization, and network details.

How to Use

Look up an IP address: Enter any IPv4 address (e.g., 8.8.8.8) and tap Lookup.

Look up a domain: Enter any domain name (e.g., google.com) and the app will resolve it to an IP address.

Find your IP: Tap "My IP Address" to discover your current public IP and location.
